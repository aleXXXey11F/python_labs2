## Коллекция объектов (Python)

### Выбранная предметная область
**Транспорт**

### Реализованный класс коллекции: **Fleet** (Автопарк)

Коллекция для хранения и управления объектами **Bus** из ЛР-1. Реализует контейнер для группы автобусов с возможностью добавления, удаления, поиска, сортировки и фильтрации.

---

### Краткое описание класса

**Fleet** — это контейнерный класс для управления коллекцией автобусов. Внутри хранит список объектов Bus.

**Основные возможности:**
- **Хранение** — список объектов Bus
- **Управление** — добавление, удаление, доступ по индексу
- **Поиск** — по номеру маршрута, имени водителя, диапазону вместимости
- **Сортировка** — по номеру маршрута, вместимости, скорости
- **Фильтрация** — получение подколлекций (на маршруте, в парке, по загрузке)
- **Итерация** — поддержка цикла for
- **Защита** — проверка типа, запрет дубликатов

---

### Методы коллекции

#### Базовые операции:
| Метод | Описание |
|-------|----------|
| `add(item)` | Добавить автобус |
| `remove(item)` | Удалить автобус |
| `remove_at(index)` | Удалить по индексу |
| `get_all()` | Получить список всех автобусов |

#### Поиск:
| Метод | Описание |
|-------|----------|
| `find_by_route_number(route_number)` | Поиск по номеру маршрута |
| `find_by_driver_name(driver_name)` | Поиск по имени водителя |
| `find_by_capacity_range(min, max)` | Поиск по диапазону вместимости |

#### Магические методы:
| Метод | Описание |
|-------|----------|
| `__len__` | Получение количества автобусов |
| `__iter__` | Итерация по автобусам |
| `__getitem__` | Доступ по индексу |
| `__contains__` | Проверка наличия автобуса |

#### Сортировка:
| Метод | Описание |
|-------|----------|
| `sort_by_route_number()` | Сортировка по номеру маршрута |
| `sort_by_capacity()` | Сортировка по вместимости |
| `sort_by_speed()` | Сортировка по скорости |
| `sort(key)` | Универсальная сортировка |

#### Фильтрация (возвращают новую коллекцию):
| Метод | Описание |
|-------|----------|
| `get_on_route()` | Автобусы на маршруте |
| `get_in_depot()` | Автобусы в парке |
| `get_by_efficiency(rating)` | Автобусы по рейтингу загрузки |
